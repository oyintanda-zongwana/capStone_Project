API Path Information

HTTP Methods Path Description
User
GET /users Gets all the users
GET /users/:id Gets a single user
POST /users/register Add a new user
POST /users/login An existing user gets a token after logging in
PATCH /users/update/:id Updates an existing user
DELETE /users/delete/:id Delete an existing user
Products
GET /products Gets all the products
GET /products/:id Gets a single product
POST /products/addProduct Add a new product
POST /products/cart Add a product to the cart
PATCH /products/update/:id Updates an existing product
DELETE /products/delete/:id Deletes existing product
